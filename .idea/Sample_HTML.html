A Sample HTML Page for Versioning

Clone
To clone an existing Git repository − Navigate to File->New->Project from Version Control->Git. Enter the repository URL, Parent directory and Directory name. Click on the clone button to continue.

Track changes
Git will track the changes that you make in repository. Let us modify any file from the repository and compare it with the repository. Navigate to VCS → Git → Compare with Latest Repository Version. The above step will open the diff window. You can see there is a new line on the right side with green background colour. Git shows it in green as we have added new contents. If we remove any contents then it’ll be shown in red colour
